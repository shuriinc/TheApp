Touch Types
A Touch type dictates what kind of interaction you’re tracking and each Touch type has a different set of fields to fit the interaction accordingly.

We have created some touch types for you, but if there are interactions you have that don’t fit into any of those categories, you can create your own.

Touch types can be anything, whether it’s a coffee date, opportunity, phone call, or any other action that helps you get business done.
  • Create your Touch type title.
  • Choose which one of Shuri’s base Touch types this interaction is most similar to. (For example, if you’re creating a Happy Hour Touch it’s probably most similar to Event). This information dictates what fields will display when this Touch type is selected.

The next time you create a new Touch, that custom Touch type will appear in the Type drop-down menu. After selecting it, the fields on the page will adjust accordingly.

For a shortcut, you can also navigate to a new Touch, and select Add New from the Type drop-down menu to follow these same steps.
