Tag Sets
Tags offer labels as a way to track trends and categories across many items and then quickly find what you’re looking for in queries. To learn more about tags, see TAGS & QUERYING.

To manually create a new tag set:
  • Click the plus sign next to Tag Sets on the Customize page.
  • Create the name of your tag set, which acts as the umbrella set your tags will live under.
  • Then, create your tags in the Add tags field and click Add New.

The next time you navigate to the appropriate contacts, you’ll be able to add these tags to their profiles by typing into the tags tab and selecting from the drop-down menu below.
