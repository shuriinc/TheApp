Tags & Querying
Tags act as labels that you can add to any person, organization or Touch in Shuri. When you want to find specific contacts or Touches quickly, run a query for certain tags and a list will display that you can take action with.

Read on to learn how to leverage this feature and keep your data organized.

A good reason to use tags:

Mark wants to track US Senators and their recent votes. In this case, he can separate the Democrats from the Republicans using tags.
  • First, he'll create a tag set titled Political Party from a Database’s customize page. For more information on how to create a tag set, see TAG SETS.
  • Then, Mark will create one tag for Democrat and one tag for Republican.
  • Back on each Senator’s profile page, he'll click to open the tags tab and type in the appropriate tag.
  • That tag will show up in the drop-down menu and Mark can select it. Now his item is labeled.

Query to easily find what you’re looking for

For Mark to see a list of only the Senators in the Democratic Party:
  • He will navigate to the People page (from the link on the bottom of the homepage).
  • Then, he'll select Query from the top navigation bar.
  • Mark will then open either of the the Tags tabs and select Democrat from within the Political Party tag set.
  • Next, he'll click Run Query and a list of only those Democratic Senators in his Database will appear.
Enhance your Query by:
  • Specifying only certain Organizations to search in.
  • Searching for people or orgs within close proximity to either your current location or another location chosen.
  • Specifying query requirements by field (ie. First Name field must contain "Robert")
Run similar queries on Organization and Touch query pages.

Do more with your data

To take action with your query results, click the 3 lines at the top righthand side of your screen. This will open a menu of different actions you can take, like creating a touch with your results, adding them to a new or existing group, selecting a subset, and more.
